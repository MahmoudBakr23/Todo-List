## Getting Started
To get a local copy of the repository please run the following commands on your terminal:
- ```$ cd <folder>```
- ```$ git clone https://github.com/MahmoudBakr23/Todo-List.git ```
- ```$ cd Todo-List ```

To run the tests:
- ```npm run test```
